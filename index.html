Hello, Bootstrap!

This is a Bootstrap website.这个类用于创建一个容器，它是 Bootstrap 栅格系统的基本布局结构。容器会自动调整其内部内容的宽度，使其在不同设备上呈现不同的布局。 Bootstrap 的网格系统是基于容器来定义的，通常一个页面只会有一个顶级容器。
